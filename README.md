# GsCartoon
## 前言
本人比较喜欢看动漫，为了追各种漫画，手机上安装了多个平台的漫画App。
所以就想自己做一个可以看各平台漫画的应用，从而诞生了该应用。
目前该应用还处于开发阶段，还有许多功能没有添加，不过已经可以浏览大部分漫画啦。
